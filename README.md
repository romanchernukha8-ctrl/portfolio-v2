## Getting Started

Clone the repository

```bash
git clone https://github.com/romanchernukha8-ctrl/portfolio-v2.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```

Deploy to GitHub Pages

```bash
npm run deploy
```
